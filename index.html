-Pasos para un algortimo de registro de alumnos

  1. Se pide el tamaño del registro de alumnos
  2. Se crea un registro con un arreglo, del tamaño especificado, y un contador en '0'
  3. Con un 'do while', se pide el nombre de los alumnos
  4. Cada nombre se guarda en registro[contador], y en cada vuelta se suma 1 al contador
  5. El bucle 'do while' se repite hasta que contador=arreglo.length

-Pasos para un algortimo de ordenamiento burbuja

  1. Se recibe un arreglo/ArrayList desordenado
  2. Con un 'for', se recorre cada elemento del arreglo/ArrayList
  3. Con un 'if', se compara cada elemento con el siguiente en posición
  4. Si el 'if' es verdadero, el valor se guarda en una variable temporal, el valor actual se reemplaza por el siguiente y el siguiente por el valor temporal
  5. Se devuelve el arreglo/ArrayList ordenado
